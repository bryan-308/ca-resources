---
title: List Style Image
description: Specify an image as the list item marker in unordered lists using the list-style-image property.
---

The **list-style-image** property used to specify an image as the list item marker in **unordered lists**.

<Tabs items={['Code', 'Output']}>
<Tab value="Code">

```html
<body>
  <h1>Most Popular Web Browsers</h1>
  <!-- [!code word:url(‘checkmarker.png‘);] -->
  <ul style="list-style-image: url(‘checkmarker.png‘);">
    <li>Mozilla Firefox</li>
    <li>Google Chrome</li>
    <li>Opera Browser</li>
    <li>Microsoft Edge</li>
    <li>Safari Browser</li>
  </ul>
</body>
```

</Tab>
<Tab value="Output">

<OrderedList listStyle="" image="/guides/lists/checkmarker.png" />

</Tab>
</Tabs>

<Callout>
Toggle light mode to see the image used in the output 😁.
</Callout>
