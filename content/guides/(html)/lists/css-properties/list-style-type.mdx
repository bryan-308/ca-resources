title: List Style Type
description: Customize the list item markers in ordered and unordered lists using the list-style-type property.
---

## Unordered List

The **list-style-type** property is used to define the style of the list item marker.

The following values below explains how to change the marker for unordered list items.

### Disc

The **disc** value sets the list item marker to a bullet.

<Tabs items={['Code', 'Output']}>
<Tab value="Code">

```html
<body>
  <h1>Most Popular Web Browsers</h1>
  // [!code word:disc]
  <ul style="list-style-type: disc;">
    <li>Mozilla Firefox</li>
    <li>Google Chrome</li>
    <li>Opera Browser</li>
    <li>Microsoft Edge</li>
    <li>Safari Browser</li>
  </ul>
</body>
```

</Tab>
<Tab value="Output">

<UnorderedList type="disc" position="inside" />

</Tab>
</Tabs>

## Ordered List

The following values sets the marker for **ordered list** items.

### Lower Alpha

The **lower-alpha** value is used to set the marker of a list item in lowercase letters.

<Tabs items={['Code', 'Output']}>
<Tab value="Code">

```html
<body>
  <h1>Phases of WDLC</h1>
  // [!code word:lower-alpha]
  <ol style="list-style-type: lower-alpha;">
    <li>Planning</li>
    <li>Analysis</li>
    <li>Design</li>
    <li>Development</li>
    <li>Testing</li>
    <li>Implementation</li>
    <li>Maintenance</li>
  </ol>
</body>
```

</Tab>
<Tab value="Output">

<OrderedList type="lower-alpha" listStyle="type" position="inside" />

</Tab>
